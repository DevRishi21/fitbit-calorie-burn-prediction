# Fitbit Workout Analytics using Machine Learning

This project analyzes Fitbit workout data using supervised and unsupervised machine learning techniques.

## Project Objectives

- Predict calories burned using regression models.
- Discover workout patterns using K-Means clustering.

## Dataset

- Fitbit Workout Dataset
- 14,102 workout records
- Features include age, gender, BMI, heart rate, MET values, workout duration, hydration, and more.

## Machine Learning Tasks

### Task 1 – Regression

Models Implemented:
- Linear Regression
- Ridge Regression
- Lasso Regression
- Decision Tree Regressor
- Random Forest Regressor
- KNN Regressor
- Support Vector Regressor (SVR)
- XGBoost Regressor (optional)

Evaluation Metrics:
- MAE
- RMSE
- R² Score

---

### Task 2 – Clustering

- Data Preprocessing
- Standard Scaling
- PCA
- K-Means Clustering
- Silhouette Score Evaluation
- Cluster Analysis

Evaluation Metrics:
- Silhouette Score
- Cluster Size Distribution
- Cluster Feature Means

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost

## Results

- Compared multiple regression models for calorie burn prediction.
- Identified workout patterns using K-Means clustering.
- Evaluated models using standard regression and clustering metrics.
